# IMS projekt

## Compile

>make

or

>make run

runs with default values

## Usage

>Usage: ./ims -a # -c # -m # [-o filename] -s # -t # -v #

-a: howitzer firing angle [°]  \
-c: drag coefficient \
-m: projectile mass [kg] \
-o: output file name \
-s: projectile cross-section area [m^2] \
-t: temperature [°C] \
-v: initial projectile velocity [m/s] 

## Plot

>make plot

for 2D graph

>make plot3d

3D graph
